# 📚 Day 2 - HTML Lists & Navigation

## 📖 Overview

On Day 2, I practiced different types of HTML lists and learned how to create and style navigation bars using HTML and CSS.

## 📂 Projects

### 📄 01_list.html
Practiced different types of HTML lists:
- Ordered List (`<ol>`)
- Reversed Ordered List
- Ordered List with `type` and `start` attributes
- Unordered List (`<ul>`)
- Definition List (`<dl>`)
- Nested Lists

### 📄 02_style_lists.html
Styled HTML lists using CSS:
- Custom bullet icon (`star.gif`)
- Styled Definition List
- Margin and Padding
- Background colors
- Rounded corners (`border-radius`)
- Dashed border
- Removed default list markers

### 📄 03_vertical_nav.html
Built a vertical navigation menu featuring:
- Block-level navigation links
- Hover effects
- Borders and background colors
- Simple sidebar-style layout

### 📄 04_Horizontal_nav.html
Built a horizontal navigation bar using:
- HTML5 `<nav>` element
- CSS Flexbox
- Hover effects
- Active navigation link
- Responsive spacing and alignment


---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## 🎯 Learning Outcomes

By completing these exercises, I learned how to:

- Create ordered, unordered, definition, and nested lists.
- Customize list styles using CSS.
- Apply spacing using margin and padding.
- Remove default list markers.
- Create horizontal navigation bars with Flexbox.
- Create vertical navigation menus.
- Apply hover effects using CSS.
- Organize HTML and CSS for clean webpage layouts.
