# 📚 Day 3 - HTML Tables & Forms

## 📖 Overview

On Day 3, I practiced creating HTML tables and forms. I learned how to organize tabular data, collect user input using different form controls, and build a complete Student Registration Form.

---

## 📂 Projects

### 📄 01_table-basics.html

Practiced HTML tables using:

- `<table>`
- `<thead>`, `<tbody>`, and `<tfoot>`
- Table rows (`<tr>`)
- Header cells (`<th>`)
- Data cells (`<td>`)
- `colspan` attribute
- Basic table styling with CSS
- `border-collapse`

---

### 📄 02_forms-basics.html

Created an HTML form containing:

- Text input
- Password input
- Search input
- Textarea
- Radio buttons
- Checkboxes
- Dropdown menu (`<select>`)
- Date picker
- `datetime-local` input
- Color picker
- File upload
- Email input with `required`
- Submit button

---

### 📄 03_group-related-controls.html

Practiced grouping related form controls using:

- `<fieldset>`
- `<legend>`
- Multiple input fields
- Organized form sections for better readability

---

### 📄 04_student-registration-form.html

Created a Student Registration Form containing:

- School logo/image
- Student Name (First Name & Last Name)
- Date of Birth
- Gender selection using radio buttons
- Address fields:
  - Street Address
  - Street Address Line 2
  - City
  - Postal / ZIP Code
  - Country (Dropdown)
- Email Address
- Phone Number
- Register button
- Reset button

---

## 🛠️ Technologies Used

- HTML5

---

## 🎯 Learning Outcomes

By completing these exercises, I learned how to:

- Create and structure HTML tables.
- Use table sections like `<thead>`, `<tbody>`, and `<tfoot>`.
- Merge table cells using the `colspan` attribute.
- Style tables using basic CSS.
- Build HTML forms using different input types.
- Use radio buttons and dropdown menus.
- Collect user information through forms.
- Create a complete Student Registration Form.
- Organize form fields for better readability and usability.
